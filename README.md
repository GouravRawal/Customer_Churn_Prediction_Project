# Customer Churn Prediction Project

## Project Overview
A Machine Learning project to predict which bank customers are likely to churn (leave the company).

## Tech Stack
- Python, Pandas, NumPy
- Scikit-learn, Matplotlib, Seaborn
- Jupyter Notebook

## Results
- 1000 customers ka data analyse kiya
- 3 ML Models trained (Logistic Regression, Random Forest, Gradient Boosting)
- Best Accuracy: 78% (Logistic Regression)
- 191 High Risk Customers identified
- Interactive Prediction UI banaya

## Project Structure
- Untitled.ipynb — Main notebook with all code
- churn_data.csv — Dataset
- churn_analysis.png — EDA Charts
- model_results.png — Model Comparison
- feature_importance.png — Feature Importance Chart

## How to Run
1. Clone this repository
2. Install requirements: pip install pandas numpy matplotlib seaborn scikit-learn
3. Open Jupyter Notebook
4. Run all cells in order
